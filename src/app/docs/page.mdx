# Pengenalan FinTale

Selamat datang di dokumentasi resmi **FinTale**! Ini adalah pusat panduan komprehensif untuk memahami seluruh mekanisme aplikasi pencatat keuangan _offline-first_ dengan elemen RPG.

## Mengapa FinTale Berbeda?

Aplikasi keuangan pada umumnya membosankan dan penuh dengan pelacakan data (tracker) yang mengancam privasi. FinTale dibangun dengan 3 pilar utama:

1. **Privasi Absolut (100% Offline)**: Data Anda tidak pernah dikirim ke _server_ mana pun. Semua disimpan secara lokal menggunakan SQLite.
2. **Gamifikasi Sehat**: Mengubah rasa malas mencatat pengeluaran menjadi kepuasan menyelesaikan _Quest_.
3. **Smart Allocation**: Sistem alokasi dana cerdas yang beradaptasi dengan _level_ finansial Anda.

> **Catatan Penting:** Karena FinTale murni berjalan _offline_, kami sangat menyarankan Anda menggunakan fitur sinkronisasi Google Drive untuk mencadangkan data secara berkala.

## Arsitektur Teknologi

FinTale terdiri dari dua repositori utama yang bersifat _Open Source_:

- **Aplikasi Mobile**: Dibangun menggunakan kerangka kerja lintas _platform_, [Flutter](https://flutter.dev).
- **Website Landing Page**: Dibangun menggunakan arsitektur web modern, [Next.js](https://nextjs.org).

Silakan telusuri menu di sebelah kiri untuk menyelami cara kerja setiap fitur, mulai dari sistem _Experience Points (XP)_, denda pelanggaran anggaran harian, hingga manajemen investasi mandiri.
